### How to use

`Ctrl+Shift+P`<br>
Pause

`Ctrl+Shift+R`<br>
Reload

`Ctrl+Shift+X`<br>
Exit

`Ctrl+Shift++`<br>

`Ctrl+Shift+Z`<br>

`Ctrl+Shift+1`<br>

doTacBatStatStrHlt
